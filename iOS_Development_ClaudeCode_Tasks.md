## 使用建议

1. **复杂任务用 `/plan`** - 先规划再执行，避免 token 浪费
2. **使用任务列表** - 大功能拆分成小任务跟踪进度
3. **善用 `/memory`** - 让 Claude 记住项目约定（如命名规范、架构模式）
4. **定期 `/cost`** - 监控 token 消耗，优化使用方式
